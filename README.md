# NewsFeed Project

## 📋 프로젝트 개요


## 🎯 주요 기능
### 1. 프로필 관리
#### 1) 프로필 조회 기능
- 다른 사용자의 프로필 조회 시, **민감한 정보는 표시되지 않습니다**.  
#### 2) 프로필 수정 기능
- 로그인한 사용자는 **본인의 사용자 정보를 수정할 수 있습니다**.  
#### 3) 비밀번호 수정 조건
- 비밀번호 수정 시, **본인 확인을 위해 현재 비밀번호를 입력**하여 올바른 경우에만 수정할 수 있습니다.
- 현재 비밀번호와 **동일한 비밀번호로는 변경할 수 없습니다**. 

### 2. 뉴스피드 게시물 관리
#### 1) 게시물 작성, 조회, 수정, 삭제 기능
- 게시물 수정, 삭제는 작성자 본인만 처리할 수 있습니다.
#### 2) 뉴스피드 조회 기능
- 기본 정렬은 생성일자 기준으로 내림차순 정렬합니다.
- 10개씩 페이지네이션하여, 각 페이지 당 뉴스피드 데이터가 10개씩 나오게 합니다.

### 3. 사용자 인증
#### 1) 회원가입 기능
- 아이디와 비밀번호, 닉네임으로 회원가입할 수 있습니다.
- 사용자 아이디는 이메일 형식이어야 합니다.
- 비밀번호는 Bcrypt로 인코딩하며, 비밀번호 형식을 지정합니다.

#### 2) 회원 탈퇴 기능
- 탈퇴 처리 시 비밀번호를 확인한 후 일치할 때 탈퇴 처리합니다.
- 탈퇴한 사용자의 아이디는 재사용할 수 없고, 복구할 수 없습니다.

### 4. 친구 관리
#### 1) 팔로우 기능
- 특정 사용자를 팔로우할 수 있습니다.
- 팔로우를 승인, 거절할 수 있습니다.

#### 2) 차단 기능
- 특정 사용자를 차단할 수 있습니다.
- 차단한 사용자 리스트를 조회할 수 있습니다.

### 5. 검색 및 정렬
#### 1) 정렬 기능
- 수정일자 최신순 정렬
- 좋아요 많은 순 정렬
#### 2) 기간별 검색 기능
- 예) 2025.11.21 ~ 2025.11.28 동안 작성된 뉴스피드 게시물 검색

### 6. 댓글
#### 1) 댓글 작성
- 사용자는 게시물에 댓글을 작성할 수 있습니다.

#### 2) 댓글 조회
- 게시글별 댓글을 조회할 수 있습니다.
  
#### 3) 댓글 수정 및 삭제
- 사용자는 본인의 댓글을 수정 및 삭제할 수 있습니다.
- 댓글 수정, 삭제는 댓글의 작성자 혹은 게시글의 작성자만 가능합니다.

### 7. 좋아요
- 사용자가 게시물이나 댓글에 좋아요를 남기거나 취소할 수 있습니다.
- 본인이 작성한 게시물과 댓글에 좋아요를 남길 수 없습니다.
- 같은 게시물에는 사용자당 한 번만 좋아요가 가능합니다.
